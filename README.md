# icar
## Minimum Requirement

Android: Version 10.0 / API Level 29  
iOS: Version 15

## Developer Notes

- This project utilize code generated by `build_runner` for `riverpod` and `freezed` implementation
- Generated freezed files `*.freezed.dart` and riverpod files `*.g.dart` is collapsed with it's parent file when opened in vscode utilizing `.vscode/settings.json`
- Several development scripts defined in `scripts` directory  

| Script  | Desscription  |
|---|---|
|`arb.sh` | Generate Localization Classes based on `/lib/src/I10n`  |
| `generator.sh`  | Generate `freezed` and `riverpod` implementation (keep this running during development)  |
| `launcher_icon.sh`  |  Generate launcher icon based on `flutter_launcher_icons.yaml` configuration |
| `splash_screen.sh`  |  Generate splash screen based on `flutter_native_splash.yaml` configuration |
| `scrcpy.sh`  |  Mirror your android phone to this device; require [scrcpy](https://github.com/Genymobile/scrcpy)|
